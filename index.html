Privacy Policy for Zeroy Clone Gmail Integration

Last updated: February 11, 2026

1. Introduction

Zeroy Clone Gmail Integration is an open-source personal productivity tool that allows users to manage their Gmail accounts through a command-line interface.

2. Data Collection and Usage

This application only accesses Gmail data that you explicitly authorize. We collect only the minimum data necessary to provide the requested functionality:

  • Email content (for reading and searching)
  • Email metadata (sender, recipient, subject, date)
  • Label and folder information

3. Data Storage

All data is stored locally on your device. No data is uploaded to any cloud servers or third-party services. Your Gmail data remains under your control at all times.

4. Data Sharing

We do not share your data with any third parties. Since this is a local application, there are no external services that receive your information.

5. Data Deletion

You can delete all local data by uninstalling the application or manually removing the configuration files. You can also revoke Gmail access at any time through your Google Account settings.

6. Open Source

This application is open source and available for inspection. You can review the code to verify how your data is handled.

7. Contact

If you have any questions about this privacy policy, please contact us through the GitHub repository issues.
