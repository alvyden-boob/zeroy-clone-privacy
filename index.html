Zeroy Clone Gmail Integration

关于本应用

Zeroy Clone Gmail Integration 是一个开源的个人生产力工具，允许用户通过命令行界面管理和自动化Gmail操作。

主要功能

  • 搜索和读取Gmail邮件
  • 发送邮件和管理草稿
  • 邮件分类和标签管理
  • 自动化重复性邮件任务

开源与安全

本应用完全开源，所有数据都存储在用户本地设备上，不会上传到任何第三方服务器。
